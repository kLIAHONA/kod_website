How to make a button look indented when clicked:


HTML Code

<div class="button">
 <a href="#" id="indented_btn">Button Text</a>
</div>
CSS Code

.button a {
    position: relative;
    text-decoration: none;
    background-color: white;
    color: black;
    padding: 5px 8px 4px 8px;
    border-radius: 5px;
    box-shadow: 2px 4px 8px 2px black;}

.button a:hover {
    text-decoration: underline;
    cursor: pointer;}

.button a:active {
    transform: translate(5px, 2px);
    box-shadow: inset 2px 2px 4px 2px black;}


How to add a gradient effect to text:


HTML Code

<p>
    Lorem ipsum <span class="gradient">gradient text</span> dolor sit amet, consectetur adipiscing elit
</p>
CSS Code

.gradient {
    background: linear-gradient(to top right, black, aqua);
    -webkit-background-clip: text;
    background-clip: text;
    -webkit-text-fill-color: transparent;
    color: transparent;
    background-size: 300%;
    animation: gradientText 3s alternate-reverse loop;}
@keyframes gradientText {
    from {
        background-position: 0%;}
    to {
        background-position: 100%;}}
Copy this CSS

.gradient {
    background: linear-gradient(/*direction*/, /*color1*/, /*color2*/);
    -webkit-background-clip: text;
    background-clip: text;
    -webkit-text-fill-color: transparent;
    color: transparent;
    background-size: 300%;
    animation: gradientText /*time*/ /*loop type*/ /*loop duration*/;}
@keyframes gradientText {
    from {
        background-position: 0%;}
    to {
        background-position: 100%;}}
Customization:


    

  • replace /*direction*/ with to right, to left, to top, to bottom, to top right, to bottom left, etc.
    • 

  • replace /*color1*/ and /*color2*/ with any color name or hex code
    note: you can add more than two color stops!
    • 

  • replace /*time*/ with time in seconds or milliseconds
    • 

  • replace /*loop type*/ with ease, linear, ease-in, ease-out, ease-in-out, alternate, alternate-reverse 
    • 

  • replace /*loop duration*/ with infinite or a number of times to loop

My Process for Designing a Website

Phase 1 – Sketching

I begin the design process for websites by figuring out how many pages it will need. For this website I needed three. Next is designing the header, footer, and navigation bar. After that it's deciding what goes on each page. Then I design each section on the page. I use an X for images and using corner marks to indicate a border or box. I also make note of what layout I want to use for each section, such as columns or carousels.



After I have a layout that I like I flesh out what the text will be and what buttons will look like and do. Once my client and I are happy with the sketches, we're ready to move onto the next phase!

Phase 2 – Rough Draft

I use my details from my sketch to begin making the outlines of the website. I make the header, footer, and navigation bar and make the buttons link to their respective HTML files. I also link a CSS file to all the pages. This has my standard codes for coloumns, font sizes that change respective to the size of the browser, as well as codes needed for everything else I put on my website. I like to have one big CSS file for the whole website rather than a separate CSS file for each HTML page, I think it makes coding easier since I know exactly where everything is.



I insert images as I go so I know what sizes they are. I work section by section to complete each page making sure to add in SEO as I go. By the end of this phase I have a website with images, layouts, and working buttons—but no colors. Now it's time to show my clients the rough draft and make any desired changes.

Phase 3 – Development

Now that I have a functional website, it's time to add colors, fonts, and the more complex codes such as pop-up windows and scrolling image frames. I work closely with my client's want-list to the find colors and fonts that match their brand identity. I also refine the layouts and make it responsive so it works on all browser sizes. By the end of this phase I have the completed website and it is ready to show to my clients!

Phase 4 – Delivery

Now it's time to upload the website and set up the site analytics. I moniter the site to make sure everything is working properly and to make any necessary changes needed. Depending on the services in each specific contract I may also update the website periodically with new content, such as blog posts and seasonal changes.
