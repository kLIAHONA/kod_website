My Process for Designing a Website

Phase 1 – Sketching

I begin the design process for websites by figuring out how many pages they need. For this website, I needed three. Next, I design the header, footer, and navigation bar. After that, it's time to decide what goes on each page. Then I design each section within the pages. I use an X for images and corner marks to indicate a border or a box. I also make note of what layout I want to use for each section, such as columns or carousels.



After I have a layout that I like I flesh out what the text will be and what buttons will look like and what they will do. Once my client and I are both happy with the sketches, we're ready to move onto the next phase!


Phase 2 – Rough Draft

I use my details from my sketch to begin making the outlines of the website. I make the header, footer, navigation bar, and buttons links. I also link a CSS file to all the pages, which has all my design code. The design code includes my standard codes for columns, font sizes that change respective to the size of the browser, as well as codes needed for everything else I put on my website. I like to have one big CSS file for the whole website rather than a separate CSS file for each HTML page, I think it makes coding easier since I know exactly where everything is.



I insert images as I go, so I know what sizes they are. I work section by section to complete each page, making sure to add in SEO as I go. By the end of this phase I have a website with images, layouts, and working buttons—but no colors. Now it's time to show my clients the rough draft and make any desired changes.

Phase 3 – Development

Now that I have a functional website, it's time to add more details! During this stage I add colors, fonts, and more complex codes such as pop-up windows and scrolling image frames. I work closely with my client's want-list to the find colors and fonts that match their brand identity. I also refine the layouts and make it responsive, so it works on all browser sizes. By the end of this phase I have the completed website and it is ready to show to my clients!

Phase 4 – Delivery

Now it's time to upload the website and set up the site analytics. I monitor the site to make sure everything is working properly and to make any needed adjustments. Depending on the services in each specific contract, I may also update the website periodically with new content. Some of the new content I add includes blog posts and seasonal changes.
